# Graph Building - Graph construction and data preparation module

Shared module for data preparation and graph construction used across training and inference pipelines.
 
## API

- `GraphBase`: Base graph buider interface
- `GraphGCN`: GCN-specific building strategy
- `minmax_scale`: Apply minmax scaling to dataset
- `groupwise_smote` : Apply SMOTE to each group of unique `edge_features` combination
